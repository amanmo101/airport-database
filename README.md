# ✈️ Airport Database Management System

A comprehensive **relational database schema** designed to simulate real-world **Airport Management** operations, fully implemented in **MySQL**.

## 📑 Report Contents

- **Entity Relationship Diagram (ERD):** Visual mapping of entities and their complex associations.
- **Relational Model:** Structured schema defining tables, primary keys, and foreign keys.
- **Normalization:** Application of database normalization techniques to ensure data integrity and reduce redundancy.
- **SQL DDL for Entities:** Complete Data Definition Language scripts for creating all database tables and constraints.
- **Mock Case Scenarios:** Real-world business questions answered through advanced **SQL Queries**.

---

## 🛠️ Implementation Details
* **Database Engine:** MySQL
* **Focus:** Schema Design, Data Integrity, and Query Optimization
